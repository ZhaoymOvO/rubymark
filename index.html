使用說明

如何將 RubyMark 引入您的網頁？

1. 引入 JavaScript 腳本

在您的 HTML 頁面中引入 rubymark.js。它會自動載入 marked.js 解析器並進行配置：

<script src="rubymark.js"></script>

2. 使用 <ruby-mark> 網頁元件（推薦）

使用標準 Custom Element 來包裹您的 Markdown 語法，內容將會自動轉換並渲染：

<ruby-mark>
# {Ruby}(振假名) 測試
這是一段包含 {旁註}(輔助說明) 的標準段落。
</ruby-mark>

3. 使用 Legacy <rubymark> 標籤

如果您想使用更簡短的標籤名稱，我們也支持 <rubymark>。即使它沒有連字元，庫中也會自動監視並將其解析渲染：

<rubymark>
此處的 {漢字}(かんじ) 同樣能被正確解析。
</rubymark>

4. 動態屬性綁定

您也可以透過 content 屬性來動態改變渲染內容：

<ruby-mark content="這是動態 {內容}(Content)"></ruby-mark>
